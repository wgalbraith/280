<h1>The Anatomy of a Dataset</h1>

<h2>Cases (Observations) and Variables</h2>
<p>
A <strong>dataset</strong> is a collection of information organized for analysis.
The individuals or objects described in the dataset are called
<strong>cases</strong> (or <strong>observations</strong>). A
<strong>variable</strong> is a characteristic recorded for each case.
</p>

<p>
For example, if a dataset contains information about students, then each
<strong>student</strong> is a case, and characteristics such as major, year (1st,2nd,etc.),
hours studied, and exam score are variables.
</p>

<hr>

<h2>Categorical and Quantitative Variables</h2>
<p>
Variables can be classified as <strong>categorical</strong> or
<strong>quantitative</strong>.
</p>

<ul>
<li><strong>Categorical variable</strong>: places a case into a group (e.g., major, year (1st,2nd,etc.)</li>
<li><strong>Quantitative variable</strong>: records a numerical value (e.g., hours studied,exam scores)</li>
</ul>

<p>
Categorical variables assign each case a group, or a catergory, while quantitative variables
assign a quantitative value to eash case. 
</p>

<hr>

<h2>Explanatory and Response Variables</h2>
<p>
When studying relationships:
</p>

<ul>
<li><strong>Explanatory variable</strong>: helps explain or predict the value of another variable</li>
<li><strong>Response variable</strong>: the outcome being studied</li>
</ul>

<p>
Example: hours studied (explanatory) and exam score (response).
</p>

<hr>

<h2>Example</h2>
<p>
Suppose a teacher collects data on students:
</p>

<ul>
<li>Each <strong>student</strong> is a <strong>case</strong></li>
<li>Variables:
  <ul>
    <li><strong>Major</strong> (categorical)</li>
    <li><strong>Hours Studied Per Week</strong> (quantitative)</li>
    <li><strong>Exam Score</strong> (quantitative)</li>
  </ul>
</li>
</ul>
<p>
If studying affects performance:
</p>

<ul>
<li><strong>Explanatory variable:</strong> Hours Studied</li>
<li><strong>Response variable:</strong> Exam Score</li>
</ul>
