<h1 id="pemdas-expression">2 PEMDAS Expression</h1>
<p>Which of these matches the expression below? (Assume your calculator
or computer uses PEMDAS.)</p>
<p><span class="math display"> \frac{1}{2} \cdot \frac{3}{4} \cdot
\frac{5}{6} </span></p>
<ol type="1">
<li>1 * 3 * 5 / 2 * 4 * 6</li>
<li>1 * 3 * 5 / (2 / 4 / 6)</li>
<li>1 * 3 * 5 / 2 / 4 / 6</li>
<li>(1 * 3 * 5) / (2 / 4 / 6)</li>
</ol>
<h1 id="pemdas-keystrokes">3 PEMDAS Keystrokes</h1>
<p>Which set of keystrokes is equivalent to this set of keystrokes?</p>
<p>1 / 7 * 2 / 4 * 5 / 6</p>
<ol type="1">
<li>1 * 2 * 5 / (7 / 4 / 6)</li>
<li>1 * 2 * 5 / (7 * 4 * 6)</li>
</ol>
<h1 id="pemdas-division">4 PEMDAS Division</h1>
<p>Which of these matches the expression below?<br />
(Assume your calculator or computer uses PEMDAS.)</p>
<p><span class="math display">
\frac{\frac{a}{b}}{\frac{c}{d}}</span></p>
<ol type="1">
<li>a * d / b / c</li>
<li>a / b / (c / d)</li>
</ol>
