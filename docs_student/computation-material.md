<h2 id="print-function">Print Function</h2>
<p>Computer functions can take many things besides numbers as input and
do many things besides return numbers as output. The print function is
very useful. Provided with text or variables, it will output things to
the screen.</p>
<pre><code>estimate = 20
print(&#39;My estimate: &#39;, estimate)

&gt; My estimate: 20</code></pre>

<h2 id="jupyter">Jupyter</h2>
<ul>
<li>Jupyter is like a word processor and spreadsheet in one program</li>
<li>By mixing words and computations together, you can clearly explain
your approach to an estimation</li>
<li>It has blocks or cells of text that are sent to the program of your
choice to be interpreted</li>
<li>Markdown: our word processor</li>
<li>Python: our spreadsheet or computation program</li>
<li>You press shift-enter to evaluate a cell</li>
</ul>
<h2 id="markdown">Markdown</h2>
<ul>
<li>You can think of this as a word processor</li>
<li>By selecting markdown as the type of cell, jupyter sends the text to
markdown to be interpreted</li>
<li>Special characters are use to tell the computer to make headings or
bold characters</li>
</ul>
<h2 id="latex">LaTeX</h2>
<ul>
<li>This is a word processor especially for math</li>
<li>By placing dollar signs ($) around some text, it is sent to LaTeX to
be interpreted</li>
<li>You can use this to write fractions and many other mathematical
symbols</li>
<li>You can find some symbols at this <a
href="http://estudijas.lu.lv/pluginfile.php/14809/mod_page/content/16/instrukcijas/matematika_moodle/LaTeX_Symbols.pdf">link</a></li>
</ul>
<p>For example</p>
<pre><code>\frac{1}{2}</code></pre>
<p>will turn into</p>
<p><span class="math display">\frac{1}{2}</span></p>

<h2 id="types">Types</h2>
<p>The computer uses ones and zeros to represent all data. Anything
stored on the computer, a poem, a photo, or a number, is represented
using ones and zeros. In order to correctly interpret the ones and zeros
the computer needs to know what you are storing.</p>
<h2 id="integers">Integers</h2>
<p>These are the counting numbers. No fractions, no decimals.</p>
<h2 id="floats">Floats</h2>
<p>These are numbers with a fractional part. If you think you are using
floats but are using integers, you may get strange results like one
divided by two equals zero.</p>
<p><a
href="https://en.wikipedia.org/wiki/Single-precision_floating-point_format">Floating
Point Example Link</a></p>
<h2 id="strings">Strings</h2>
<p>This is how the computer stores words.</p>
<p><a href="https://en.wikipedia.org/wiki/ASCII">ASCII Example
Link</a></p>
<h2 id="dates-and-times">Dates and Times</h2>
<p>Sometimes the computer stores numbers that represent dates and times.
These formats can be subtle, so be sure you know how they work when
using them.</p>
