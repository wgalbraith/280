<h1 id="draw-a-histogram">2.2 Draw a Histogram</h1>
<p>Draw a histogram using data that you find online. Find a data source
with individual observations (you can start with the list below).</p>
<p>Before you begin, state</p>
<ul>
<li>What is the x-axis description and units</li>
<li>What is the y-axis description and units</li>
<li>What does each dot represent?</li>
</ul>
<p>When you have completed a histogram</p>
<ul>
<li>Tell the size of your bins</li>
<li>Discuss how you chose the size of your bins</li>
<li>Be sure the x and y axes are clearly labeled</li>
<li>You can create bars or “X”s to show the height of the bars in the
histograms.</li>
<li>Be sure it is clear what each element in the data represents (state,
CSU campus, other)</li>
<li>Estimate the mean, median, 25th, and 75th percentiles</li>
</ul>
