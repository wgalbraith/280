<h1 id="counting">1 Counting</h1>
<p>One a piece of paper, “count” in base 8 over the following ranges and
show the corresponding base 10 numbers to the side. You may find a
number line useful for this activity.</p>
<ul>
<li>the base 10 numbers 0 to 24 (“0” to “30” in base 8)</li>
<li>the base 10 numbers 56 to 72 (“70” to “110” in base 8)</li>
</ul>
<h1 id="base-5-to-base-10">5 Base 5 to Base 10</h1>
<p>The number 1324 is a number in base 5, i.e <span class="math inline">$1324_5$. Write this number in base 10?</p>
<p>Challenge: the number 12.34 is a number in base 5, what is it in
base 10? (Hint: Numbers to the right of the decimal point in base 5 follow the same pattern as our base 10)</p>
