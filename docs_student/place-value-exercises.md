<h1 id="counting">1 Counting</h1>
<p>One a piece of paper, “count” in base 8 over the following ranges and
show the corresponding base 10 numbers to the side. You may find a
number line useful for this activity.</p>
<ul>
<li>the base 10 numbers 0 to 24 (“0” to “30” in base 8)</li>
<li>the base 10 numbers 56 to 72 (“70” to “110” in base 8)</li>
</ul>
<h1 id="making-change-10">2 Making Change 10</h1>
<p>You need to give someone 43 cents using only dimes and pennies and
using the least total number of coins possible.</p>
<ul>
<li>How many pennies?</li>
<li>How many dimes?</li>
</ul>
<h1 id="making-change-5">3 Making Change 5</h1>
<p>You are going to give someone 43 cents, this time only using pennies,
nickels, and quarters, again using the least number of coins.</p>
<ul>
<li>How many pennies?</li>
<li>How many nickels?</li>
<li>How many quarters?</li>
</ul>
<h1 id="making-change-3">4 Making Change 3</h1>
<p>Now imagine coins where still have pennies, but other coins are worth
3 times more than the next lowest coin. You want to give someone 43
cents.</p>
<ul>
<li>What are the values of the coins?</li>
<li>How many of each coin type do you have?</li>
</ul>
