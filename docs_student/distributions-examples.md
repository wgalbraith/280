<h1 id="distributions">Histograms to Distributions</h1>
<p>So far we have been looking at histograms, which contain the measured
data for a collection of objects or observations.</p>
<p>Often we can construct a mathematical model of these histograms that
allow us to reason and make predictions.</p>
<p>These mathematical models are called distributions.</p>
